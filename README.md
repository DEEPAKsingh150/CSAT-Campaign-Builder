✨ Features
📝 Content Tab
Initial Feedback – editable title and subtitle.

Feedback Page

Rating type toggle (⭐ Stars or 🔢 Numbers 1‑5).

Dynamic Options – add or delete custom feedback options (e.g., "Excellent", "Good").

Additional comment toggle (enable/disable textarea).

Customizable submit button text.

Thank You Page

Upload media (supports PNG, JPG, JPEG, GIF – also Lottie via JSON, but simplified to image formats).

Title, subtitle, and button text.

🎨 Styling Tab
Colors – background, title, subtitle, button, button text, rating selected/unselected.

Typography & Sizing – font size, font weight, border radius, button width & height.

All changes reflect in the preview in real time – no save or refresh needed.

📱 Live Mobile Preview
Accurate phone‑frame simulation.

Interactive flow: rating → feedback form → thank‑you page.

Instantly updates on every change made in the content or styling panels.

🛠️ Tech Stack
React 18 – UI library with hooks and context API.

Vite – build tool and development server.

Tailwind CSS – utility‑first styling (with custom CSS for the phone frame).

JavaScript (ES6+) – all components are functional.

Netlify / Vercel – deployment (choose your preferred platform).
